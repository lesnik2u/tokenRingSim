# How to install and run

## Prerequisites

You need to install:

- Cmake
- Ninja

### Mac/linux

1. Clone the repository

2. Run 'cmake -B build -G Ninja' in the root folder

3. cd into build folder and run 'ninja'.

4. Execute generated executable
